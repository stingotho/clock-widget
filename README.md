# Clock Widget

A customizable, frameless, transparent clock widget for Windows 11 built with Electron.

![Clock Widget](icon.png)

## Features
- **Customizable Frames**: Round, Square, Glass.
- **Visual Styles**: Digital, Minimal, Retro.
- **Colors**: Custom text and background colors.
- **Draggable**: Move it anywhere on your screen.
- **Always on Top**: Keep it visible above other windows.
- **System Tray**: Minimize to tray and control from there.

## Installation

### For Users (Download)
1. Go to the [Releases](https://github.com/stingotho/clock-widget/releases) page.
2. Download the `ClockWidget-win32-x64.zip` file.
3. Extract the zip file.
4. Run `ClockWidget.exe`.

### For Developers (Build from Source)
1. Clone the repository:
   ```bash
   git clone https://github.com/stingotho/clock-widget.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm start
   ```
4. Build the app:
   ```bash
   npm run build
   ```
